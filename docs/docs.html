This API allows user registration, login, and secure password storage with end-to-end encryption.

Auth Endpoints

POST /api/auth/signup

Registers a new user with username, email, and password.

{
  "username": "joel",
  "email": "joel@example.com",
  "password": "StrongPass123!"
}

{
  "message": "User registered successfully",
  "userId": "64f4a7f0fbdcf01a3e923abc"
}
POST /api/auth/login

Logs in a user using email and password.

{
  "email": "joel@example.com",
  "password": "StrongPass123!"
}

{
  "message": "Login successful",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6..."
}

Vault Endpoints

POST /api/vault/store

Saves an encrypted password entry for a site.

{
  "userId": "abc123",
  "site": "gmail.com",
  "password": "gmail@123"
}

{
  "message": "Password saved successfully"
}
GET /api/vault/list/:userId

Returns all saved password entries for a user.

[
  {
    "site": "gmail.com",
    "password": "gmail@123"
  },
  {
    "site": "facebook.com",
    "password": "fb@pass456"
  }
]
